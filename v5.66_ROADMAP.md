# v5.66 Development Roadmap: Architectural Hardening 🛠️

This document tracks the progress of architectural improvements and reliability fixes identified in the v5.65 audit. The goal is to eliminate "silent" failures, race conditions, and memory safety issues.

---

## ✅ Phase 1: Critical System Safety (Data Corruption & Deadlocks)
*Status: COMPLETED* 🚀

- [x] **Task 1.1: SPIFFS Mutex Lockout** 🛡️
    - **Issue:** Concurrent SPIFFS access by GPRS and Scheduler tasks can lead to filesystem corruption.
    - **Action:** Wrap ALL file operations in `fsMutex`.
    - **Status:** Done (v5.66)

- [x] **Task 1.2: Cross-Core Volatile Visibility** ⚡
    - **Issue:** Shared flags without `volatile` qualifier were causing core desyncs.
    - **Action:** Add `volatile` to all shared flags.
    - **Status:** Done (v5.66)

- [x] **Task 1.3: Recursive Mutex Safety in I2C Recovery** 🚫
    - **Issue:** `recoverI2CBus()` causing deadlocks if called while holding `i2cMutex`.
    - **Action:** Refactor `recoverI2CBus` to handle nested locks.
    - **Status:** Done (v5.66)

---

## ⚙️ Phase 2: Hardware & Memory Alignment 
*Status: COMPLETED* 🚀

- [x] **Task 2.1: GPIO 34/35 Hardware Pull-ups Validation** 🌧️
    - **Action:** Stripped misleading internal pullups. Documented mandatory external 10k requirement.
- [x] **Task 2.2: RTC_DATA_ATTR Type Sync** 📜
    - **Action:** Sub-task implicitly resolved during ODR Phase 3 Hoist.
- [x] **Task 2.3: ULP overflow Sentinel** 🛰️
    - **Action:** Removed destructive `wind_count` > 65000 reset; scheduler naturally handles single 16-bit wrap.

---

## ✅ Phase 3: Architectural Modernization (ODR Structural Fix)
*Status: COMPLETED (Structural Part)* 📦

- [x] **Task 3.1: The globals.h Refactor (ODR Fix)** 📦
    - **Issue:** Initialized variables in header caused linker warnings and potential memory overlaps.
    - **Action:** Move all initializers to `AIO9_5.0.ino`.
    - **Status:** Done (v5.66)

- [x] **Task 3.2: I2C Measurement Window Protection** 🌡️
    - **Action:** Refactored `readHDC` to boldly hold `i2cMutex` continuously across `vTaskDelay` to enforce analog measurement silence.
- [ ] **Task 3.3: ISR-Safe strings (strftime)** 📋
    - **Status:** Backlog
