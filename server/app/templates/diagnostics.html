📦 Stack Watermarks

  • ■ ≥ 512 bytes — Safe
  • ■ 256–511 bytes — Watch (headroom shrinking)
  • ■ < 256 bytes — Critical (overflow risk)
  • — — No Data (old firmware)

Tracks the all-time minimum free stack bytes per task since last reboot. Persistent via RTC RAM across deep sleep.

📡 Signal & Network

  • ■ RSSI ≥ −75 dBm — Strong
  • ■ −90 to −75 dBm — Marginal
  • ■ < −90 dBm — Weak / Dropping

Min/Max tracks the worst and best signal seen during all HTTP transmission windows in the current boot session.

🌡 I2C & Modem Temp

  • ■ Temp ≤ 50°C — Normal
  • ■ 51–60°C — Elevated (check enclosure)
  • ■ > 60°C — Hot (thermal throttling risk)
  • I2C Errors > 0 — Bus contention or loose sensor cable.
